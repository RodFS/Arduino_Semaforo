# Semáforo

Pequeno projeto desenvolvido para aprendizagem de manuseio da placa Arduino e seus componentes, além da programação do mesmo.
Um simples projeto de um semáforo, onde foi utilizado:

- Placa Arduino UNO
- 3 LEDs
- 3 resistores 330 ohms
- Protoboard
